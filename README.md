# Moody
## 역할

- 이미지 자동 추천
  - 자연어 감정 분석 : 박찬희
  - 이미지 감정 분석 : 박예지
- 데이터베이스
  - Firebase : 박찬희, 홍수희
  - SQLite : 김민영
- 안드로이드 스튜디오 : 김민영, 홍수희, 박예지, 박찬희

## 제작 기간
- 2020.04 ~ 2020.09

## About
- 실시간 대화 중, 이모티콘이나 이미지를 찾다 대화 흐름이 끊기게 되는 경우가 발생한다.
- 흐름이 끊기지 않고, 텍스트를 전송했을 때 바로 관련 이미지를 추천하는 기능이 있으면 편리할 것이다.
- 채팅화면에서 이미지를 자동으로 추천해주는 앱을 구상했다.
